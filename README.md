# Woody
**Woody** 简洁大气、安全可靠的书签管理工具

![woody](arts/icon.png)

### Woody
**Woody** 是一个轻量的谷歌浏览器插件（事实上叫扩展更恰当一些），谷歌应用商店有很多书签类插件，但是不是离线的，就是api是私有服务器的，不敢保证数据安全与服务稳定。 **Woody** 采用 [Github Gist](https://gist.github.com/) 作为数据同步接口，可保证数据安全，且只要你通过 **Chrome** 安装 **Woody**，你就可以随时随地管理和浏览你的书签。
